# API仕様書

## 基本情報

| 項目 | 内容 |
|------|------|
| ベースURL | `http://localhost:8080` |
| データ形式 | JSON |

## エンドポイント一覧

| # | メソッド | パス | 説明 |
|---|----------|------|------|
| 1 | GET | /api/tasks | タスク一覧取得 |
| 2 | POST | /api/tasks | タスク新規登録 |
| 3 | PUT | /api/tasks/{id} | タスク更新（完了状態の切り替え） |
| 4 | DELETE | /api/tasks/{id} | タスク削除 |

---

## 1. タスク一覧取得

**GET** `/api/tasks`

### レスポンス `200 OK`

```json
[
  {
    "id": 1,
    "title": "買い物をする",
    "done": false,
    "createdAt": "2026-03-23T10:00:00"
  },
  {
    "id": 2,
    "title": "本を読む",
    "done": true,
    "createdAt": "2026-03-23T11:00:00"
  }
]
```

---

## 2. タスク新規登録

**POST** `/api/tasks`

### リクエストボディ

```json
{
  "title": "買い物をする"
}
```

### レスポンス `201 Created`

```json
{
  "id": 1,
  "title": "買い物をする",
  "done": false,
  "createdAt": "2026-03-23T10:00:00"
}
```

---

## 3. タスク更新

**PUT** `/api/tasks/{id}`

### パスパラメータ

| パラメータ | 型 | 説明 |
|------------|-----|------|
| id | Long | 更新対象のタスクID |

### リクエストボディ

```json
{
  "title": "買い物をする",
  "done": true
}
```

### レスポンス `200 OK`

```json
{
  "id": 1,
  "title": "買い物をする",
  "done": true,
  "createdAt": "2026-03-23T10:00:00"
}
```

---

## 4. タスク削除

**DELETE** `/api/tasks/{id}`

### パスパラメータ

| パラメータ | 型 | 説明 |
|------------|-----|------|
| id | Long | 削除対象のタスクID |

### レスポンス `204 No Content`

レスポンスボディなし
