# 要件定義書

## プロジェクト概要

| 項目 | 内容 |
|------|------|
| プロジェクト名 | Todoアプリ |
| 作成日 | 2026-03-23 |
| 技術スタック | Spring Boot 3.5.12 / Java 21 / MySQL 8.0 / HTML + CSS + Vanilla JS |

## 機能要件

| # | 機能 | 説明 |
|---|------|------|
| 1 | タスク一覧表示 | 登録済みのタスクを一覧で表示する |
| 2 | タスク追加 | タイトルを入力してタスクを新規登録する |
| 3 | 完了チェック | チェックボックスでタスクの完了/未完了を切り替える |
| 4 | タスク削除 | タスクを削除する |

## 非機能要件

- フロントエンドはVanilla JS（将来的にReact移行を見据えた設計）
- バックエンドはREST APIとして実装し、フロントエンドとはfetch APIで通信
- データベースはMySQL

## 対象外

- ユーザー認証・ログイン機能
- タスクの期限・優先度設定
- タスクの編集機能
