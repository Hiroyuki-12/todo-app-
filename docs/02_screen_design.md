## タスク一覧画面

### レイアウト概要

```
+----------------------------------+
|         Todoアプリ                |
+----------------------------------+
| [タスクを入力...]  [追加]          |
+----------------------------------+
| □ タスクA                [削除]   |
| ☑ タスクB（完了済み）    [削除]   |
| □ タスクC                [削除]   |
+----------------------------------+
```

### 構成要素

| 要素 | 種類 | 説明 |
|------|------|------|
| タイトル入力欄 | `<input type="text">` | タスク名を入力 |
| 追加ボタン | `<button>` | 入力内容をAPIに送信して追加 |
| タスク一覧 | `<ul>` | タスクの一覧を表示 |
| 完了チェックボックス | `<input type="checkbox">` | チェックでdone=trueに更新 |
| 削除ボタン | `<button>` | 対象タスクをAPIで削除 |

### 画面の動作フロー

1. 画面読み込み時 → `GET /api/tasks` でタスク一覧を取得して表示
2. 追加ボタン押下 → `POST /api/tasks` でタスクを登録、一覧を再描画
3. チェックボックス変更 → `PUT /api/tasks/{id}` でdoneを更新、一覧を再描画
4. 削除ボタン押下 → `DELETE /api/tasks/{id}` でタスクを削除、一覧を再描画
